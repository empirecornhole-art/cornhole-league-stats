# League Stats

Cornhole league Progressive Web App.

## First setup

1. Add the files in this project to GitHub.
2. In Vercel, connect the GitHub repo.
3. Add environment variable `ADMIN_PASSWORD`.
4. Add Vercel Blob storage to the project.
5. Upload the league workbook from `/admin`.

## Pages

- `/` Dashboard
- `/players` Player stats
- `/weeks` Weekly results
- `/compare` Player comparison
- `/admin` Workbook upload
